MMO RPG

MMORPG is a project that I am currently working on. I started this project to improve coding skills while creating something I love. MMMORPG are super big and it not something you just do alone. Thats why in this project I step by step create systems like skill system, combat system, inventory system etc. On top of that I work on terrain for the map with lightning, fog etc. And I might also do some 3D modeling if I really want a break from it all.

Omae Wa Gandalf

Gandalf is a top down rogue like dungeon crawler. The game was made in Unreal Engine and is mostly c++ based. So no blueprints. This project was all about program challenges. Each run the level is proceduraly generated. Which was my main challenge.

The Mofflys

The Mofflys is a vr shooter. In the game you are a creation of the mofflys, 'murder clone'. With a wide variety of weapons you will play through sets of minigames to invade earth and cause mayhem. The game exists of multiple worlds with classic themes like Egypt, Amazon River, Western world and more.

Space Escape

Space Escape is a 3D platformer which is based around the element challenge. In this game you are trapped in space. And you have to find your way through the asteroids by collecting currency with which you can purchase powerups at the vendor machine.

Orbit Singularity

Orbit Singularity is an orbit game created during the 2018 Global Gamejam. You try to find extraterrestial life by sending signals through as many satellites as possible. In the game you have to manage satilites in different orbit lanes. Only satilites in the outer ring will be send of and grant you points. In interval new satilites will be send into the orbit from earth. Manage them well or they will end up crashing.
