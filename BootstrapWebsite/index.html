Conquer

Conquer will be a 3D MMORPG. The main idea behind this game is a combat system based around apm(actions per minute). Skill improvements are gained by practise. In addition gear and skills within the game are all part of a craft system. Conquer will focus on creating a fun experience in both pve and pvp. Grinding away must be fun. This is something that I personally find lacking in current mmo's.

Status: Unstarted
