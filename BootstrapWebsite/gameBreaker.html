Game Breaker

2D | Top Down | Puzzle | Dungeon Crawler

Game Breaker is a puzzle based top down dungeon crawler. In the game the main caracter wants to break out of the game. But the developer of the game wich is the bad guy tries to stop you. Solve puzzles, bypass traps, stay alive and you'll be victorious. A short gameplay video can be found above.

Who, What, Where?

Game breaker was created during a project in my second school year. Together with four classmates I made Game Breaker. Game breaker was my first project using Unity. Like a global gamejam this game was also created based on a word, 'impotence'. However we were allowed to step away from this word. It was there to assist and guide the prototype phase of the project. So in the end our final product isn't all about impotence.

Wanna read our Game Design Document? This is only possible on request and approval by my team and me.
